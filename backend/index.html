Welcome to Scanned exams

This app imports exams from the scanning system Windream to Canvas. The app will walk you through these steps:

  • Authentication
  • Create assignment
  • Upload scanned exams to assignment

For the app to work, the course needs to be published but the grade posting policy is set to ”Manual” which means that you can grade all exams and then publish the grades for all students at the same time. More info about grading posting policies can be found here. If not all exams have been scanned in Windream when you run the app for the first time, you can run the app again to import the added exams. Make sure to turn off the ”All submissions” notifications on the course, otherwise you’ll get an individual mail for each student submission uploaded by the app.

If you have any questions, or any problem arises, please contact it-support@kth.se!
